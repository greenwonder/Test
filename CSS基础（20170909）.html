什么是CSS

Cascading Style Sheets（CSS）指定文档如何呈现

  • 网页背景色，背景图片，
  • 文本颜色
  • 文本，图片的显示位置

为什么用CSS

CSS规则

基本规则

  • 选择符(Selector)

    选择一个或多个样式元素

  • 声明(Declaration)

    声明一个规则，指定样式元素的属性

  • 属性

  • 属性值

    冒号后面，分号结尾

选择符(Selector)

  • 多个样式元素
    p,li,h1{
  color: red;
}

选择符类型

元素

html的元素

ID

指定ID的元素（ID唯一) 

#my-id
选择:<p id="my-id">或<a id="my-id">

Class

指定Class的元素(不唯一) 

.my-class
选择:<p class="my-id">或<a class="my-id">

属性

页面上拥有指定属性的元素 

img[src]
选择:<img src="myimage.png>而不是<img>

伪类

指定的元素，但是需要在特殊的状态 

a:hover
选择<a>,但是只在鼠标悬停在链接上时。

Box Model

margin

外边距

四种位置表示

  • 1个值

    所有四边

  • 2个值

    垂直|水平

  • 3个值

    上|水平|下

  • 4个值

    上|右|下|左

顺时针

数值单位

  • px

    像素

  • em

    1em 表示1个父元素字体的高度

  • %

    相对于父元素的百分比

  • rem

    1rem 表示1个根元素的字体高度

  • auto

    水平居中

** padding 内边距

width

元素宽度

background-color

背景色

color

文字的颜色

text-shadow

text-shadow: 2px 2px 1px black;
  • 第一个像素值设置了水平方向的阴影值
  • 第二个像素值设置了垂直方向的阴影值
  • 第三个像素值设置了阴影模糊的距离（越大的值表示越模糊）
  • 第四个值设置了阴影的颜色
